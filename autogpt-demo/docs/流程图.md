# AutoGPT Demo —— 流程图
布道者

## 1. 整体执行流程

```
                      ┌─────────────────┐
                      │   用户输入目标    │
                      └────────┬────────┘
                               │
                ┌──────────────▼──────────────┐
                │   Phase 1: Plan (规划)       │
                │  ┌───────────────────────┐  │
                │  │ 分析目标语义            │  │
                │  │ 调用 LLM 或 规则引擎    │  │
                │  │ 生成子任务列表          │  │
                │  └───────┬───────────────┘  │
                └──────────┼──────────────────┘
                           │
                           ▼
              ┌────────────────────────────┐
              │  tasks = [                 │
              │    {title, desc, tool,     │
              │     criteria},             │
              │    {title, desc, tool,     │
              │     criteria},             │
              │    ...                     │
              │  ]                         │
              └────────────┬───────────────┘
                           │
          ┌────────────────▼─────────────────┐
          │  Phase 2: Execute (自主执行循环)    │
          │                                   │
          │  ┌─────────────────────────────┐  │
          │  │  for task in tasks:         │  │
          │  │      ┌─────────────────┐    │  │
          │  │      │  Think (思考)    │◄───┤  │
          │  │      │  分析上下文       │    │  │
          │  │      │  推理最佳行动     │    │  │
          │  │      └───────┬─────────┘    │  │
          │  │              │              │  │
          │  │              ▼              │  │
          │  │      ┌─────────────────┐    │  │
          │  │      │  Decide (决策)   │    │  │
          │  │      │  选择工具和参数   │    │  │
          │  │      └───────┬─────────┘    │  │
          │  │              │              │  │
          │  │              ▼              │  │
          │  │      ┌─────────────────┐    │  │
          │  │      │   Act (执行)     │    │  │
          │  │      │  调用工具        │    │  │
          │  │      └───────┬─────────┘    │  │
          │  │              │              │  │
          │  │              ▼              │  │
          │  │      ┌─────────────────┐    │  │
          │  │      │ Observe (观察)   │    │  │
          │  │      │ 分析结果         │    │  │
          │  │      │ 更新记忆         │    │  │
          │  │      └───────┬─────────┘    │  │
          │  │              │              │  │
          │  │      ┌───────▼──────────┐   │  │
          │  │      │ 子任务完成?        │   │  │
          │  │      │ 达成最大迭代?      │   │  │
          │  │      └──┬────────────┬──┘   │  │
          │  │     是  │            │ 否    │  │
          │  │   ┌─────▼─┐      ┌──▼───┐  │  │
          │  │   │ 下一个 │      │回到   │──┘  │
          │  │   │ 子任务 │      │Think  │     │
          │  │   └───────┘      └──────┘     │
          │  └─────────────────────────────┘  │
          └───────────────┬───────────────────┘
                          │
                          ▼
              ┌───────────────────────────┐
              │  Phase 3: Summary (总结)    │
              │  汇总所有执行记录            │
              │  生成结构化报告              │
              └───────────┬───────────────┘
                          │
                          ▼
              ┌───────────────────────────┐
              │  Phase 4: Persist (持久化)  │
              │  存储到 LongTermMemory     │
              │  保存检查点                │
              └───────────┬───────────────┘
                          │
                          ▼
                 ┌─────────────────┐
                 │   返回最终报告    │
                 └─────────────────┘
```

## 2. Think-Act-Observe 循环详解

```
Enter Think-Act-Observe
        │
        ▼
┌─────────────────────────────────────────────────────────┐
│                     THINK (思考)                         │
│                                                         │
│  输入:                                                   │
│    · 当前目标 (WorkingMemory.goal)                        │
│    · 当前子任务 (WorkingMemory.plan[current_index])       │
│    · 对话历史 (ShortTermMemory.get_recent(10))           │
│    · 历史经验 (LongTermMemory.search(goal))              │
│    · 执行进度 (WorkingMemory.get_progress())             │
│                                                         │
│  处理:                                                   │
│    LLM 模式: system_prompt + context → LLM 推理           │
│    模拟模式: 根据任务类型 + 执行步数 → 规则推理             │
│                                                         │
│  输出:                                                   │
│    THOUGHT: 自然语言推理过程                             │
│    ACTION: 工具名 (web_search / code_exec / ...)        │
│    ARGUMENTS: {"param": "value"}                        │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                     DECIDE (决策)                         │
│                                                         │
│  解析 LLM 输出:                                          │
│    action = re.search(r'ACTION:\s*(\w+)', thought)      │
│    args   = re.search(r'ARGUMENTS:\s*({.*})', thought)  │
│                                                         │
│  验证:                                                   │
│    - 工具是否已注册？                                     │
│    - 参数是否符合 Schema？                               │
│    - 是否满足安全策略？                                   │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                      ACT (执行)                           │
│                                                         │
│  ToolRegistry.execute(action, **args)                    │
│    │                                                    │
│    ├── web_search →  DuckDuckGo / 模拟搜索               │
│    ├── code_exec  →  subprocess 子进程                   │
│    ├── file_ops   →  workspace 文件读写                  │
│    ├── web_browse →  HTTP 抓取 + HTML 解析               │
│    └── finish     →  任务完成                            │
│                                                         │
│  返回: ToolResult(success, content, metadata)            │
└──────────────────────────┬──────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                    OBSERVE (观察)                         │
│                                                         │
│  分析结果:                                               │
│    if result.success:                                    │
│        → 提取关键信息                                    │
│        → 判断子任务是否完成                               │
│        → 如完成，推进到下一个子任务                        │
│    else:                                                 │
│        → 分析失败原因                                    │
│        → 决定重试或换策略                                │
│                                                         │
│  更新记忆:                                               │
│    WorkingMemory.log_action(thought, action, result)     │
│    ShortTermMemory.add_agent(...)                        │
│    ShortTermMemory.add_tool_result(...)                  │
│                                                         │
│  条件判断:                                               │
│    ├── 子任务完成 → 下一个子任务                          │
│    ├── 全部完成   → Summary phase                        │
│    ├── 失败重试   → 回到 THINK                           │
│    └── 达到上限   → Summary phase                        │
└─────────────────────────────────────────────────────────┘
```

## 3. 记忆系统数据流

```
                   ┌──────────────────────────┐
                   │     WorkingMemory        │
                   │                          │
                   │  goal: "..."             │
                   │  plan: [Task×4]          │
                   │  current_index: 1        │
                   │  execution_log: [Step×3] │
                   └───────────┬──────────────┘
                               │
        ┌──────────────────────┼──────────────────────┐
        │ 读取                 │ 写入                  │ 持久化
        ▼                      ▼                      ▼
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│ ShortTerm    │    │ ShortTerm    │    │ LongTerm     │
│ Memory       │    │ Memory       │    │ Memory       │
│              │    │              │    │              │
│ Think 时:    │    │ 每次 Act 后: │    │ 任务结束后:   │
│ get_recent() │    │ add_agent()  │    │ store()      │
│ 返回最近 N 轮│    │ add_tool()   │    │ 向量化 + 索引 │
└──────────────┘    └──────────────┘    └──────────────┘
        │                                      │
        │ 提供上下文                            │ 语义搜索
        ▼                                      ▼
┌──────────────────────────────────────────────────┐
│              THINK (LLM Context)                  │
│                                                   │
│  System: 你是 {name}，你可以调用以下工具...        │
│  History: [最近 10 轮对话]                        │
│  Memory: [语义搜索到的历史经验, 最多 3 条]          │
│  Progress: 当前进度 (2/5)                         │
│  Instruction: 分析状态，决定下一步...              │
└──────────────────────────────────────────────────┘
```

## 4. 工具注册与调用

```
┌──────────────────────────────────────────────────────────┐
│                    ToolRegistry                          │
│                                                         │
│  _tools = {                                             │
│    "web_search":  WebSearchTool(),                      │
│    "file_ops":    FileOperationsTool(),                 │
│    "code_exec":   CodeExecutionTool(),                  │
│    "web_browse":  WebBrowseTool(),                      │
│  }                                                      │
│                                                         │
│  register(tool):         添加到字典                      │
│  get(name):              O(1) 查找                      │
│  execute(name, **args):  查找 → 执行 → 返回 ToolResult   │
│  get_commands_description(): 生成 LLM 可用的命令列表      │
└──────────────────────────────────────────────────────────┘
```

## 5. LLM 模式 vs 模拟模式分支

```
                  ┌─────────────┐
                  │  run(goal)  │
                  └──────┬──────┘
                         │
              ┌──────────▼──────────┐
              │ use_live_llm?        │
              └──────┬──────┬───────┘
                     │      │
              LLM=是 │      │ LLM=否
                     │      │
    ┌────────────────▼─┐  ┌─▼────────────────┐
    │ _plan_with_llm   │  │ _plan_with_rules │
    │ 真实 LLM 推理     │  │ 关键词匹配+模板   │
    └──────────────────┘  └──────────────────┘
                     │      │
    ┌────────────────▼─┐  ┌─▼────────────────┐
    │ _think_with_llm  │  │ _think_simulated │
    │ LLM 自由推理      │  │ 步骤计数器推理    │
    └──────────────────┘  └──────────────────┘
                     │      │
    ┌────────────────▼─┐  ┌─▼────────────────┐
    │ 解析 LLM 输出     │  │ 规则决策          │
    │ THOUGHT/ACTION/   │  │ 任务类型→工具选择  │
    │ ARGUMENTS 格式    │  │ 进度→操作参数      │
    └──────────────────┘  └──────────────────┘
```

## 6. 正常执行 vs 异常处理

```
正常路径:
  Think → Decide → Act → Observe → (子任务完成) → 下一个 → ...
                                 → (全部完成)   → Summary → Persist

错误处理:
  Think → Decide → Act → ❌ 错误发生
                              │
                          ┌───▼──────────┐
                          │ 分析错误原因   │
                          │ 记录到日志     │
                          └───┬──────────┘
                              │
              ┌───────────────┼───────────────┐
              │               │               │
         可重试错误      不可重试错误      致命错误
         (超时/网络)    (权限/格式错误)   (系统崩溃)
              │               │               │
         回到 Think      跳过子任务      终止执行
         调整策略         记录失败        保存检查点
              │               │               │
              └───────────────┴───────────────┘
                              │
                         进入 Summary 阶段
```
