# AutoGPT Demo —— 测试说明
布道者

## 测试策略

本项目的测试采用**金字塔模型**：

```
         /\
        /整\         集成测试 (2-3 个)
       /合测\
      /──────\
     / 功能测试 \     功能验证测试 (8-10 个)
    /────────────\
   /   单元测试    \   工具/内存/Agent 单元测试 (20+ 个)
  /────────────────\
```

## 测试结构

```
tests/
└── test_autogpt.py
    ├── TestTools          # 工具层测试
    ├── TestMemory          # 记忆系统测试
    ├── TestAgentCore       # Agent 核心循环测试
    ├── TestIntegration     # 集成测试
    └── TestConfig          # 配置模块测试
```

## 运行测试

```bash
# 运行全部测试（无需 API Key）
pytest tests/ -v

# 仅运行单元测试
pytest tests/ -v -k "not Integration"

# 运行特定测试类
pytest tests/test_autogpt.py::TestTools -v
pytest tests/test_autogpt.py::TestMemory -v
pytest tests/test_autogpt.py::TestAgentCore -v

# 带覆盖率报告
pip install pytest-cov
pytest tests/ -v --cov=src --cov-report=html
```

## 编写新测试

```python
# 工具测试模板
def test_your_new_tool():
    tool = YourTool()
    result = tool.execute(param="value")
    assert result.success is True
    assert "expected" in result.content

# Agent 测试模板
def test_agent_with_custom_goal():
    config = AgentConfig(max_iterations=3)
    agent = AutoGPTAgent(config=config)
    result = agent.run("你的测试目标")
    assert result["goal"] == "你的测试目标"
    assert len(result["plan"]) >= 2
    assert result["iterations"] > 0

# 参数化测试
@pytest.mark.parametrize("goal,expected_tasks", [
    ("调研 Python 最佳实践", 4),
    ("开发一个应用", 4),
    ("计算一些数据", 3),
    ("做点什么", 3),
])
def test_plan_various_goals(goal, expected_tasks):
    config = AgentConfig()
    agent = AutoGPTAgent(config=config)
    result = agent.run(goal)
    assert len(result["plan"]) == expected_tasks
```
